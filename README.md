# Simulations-for-Solenoid-Magnetic-Field

论文《不同构型螺线管漏磁特性讨论以及磁场构型的数值模拟》中数值模拟所用的源代码。

函数emploar.m的作用是画出给定横截面位置直螺线管磁场的向量场图。在函数的参数中，R代表直螺线管半径，I为电流，L为长度，l为所画截面距离螺线管一端的高度，d为螺距，N为取直螺线管的总电流元数，N1为采样点的个数的平方根，rm、rM为采样点与轴心距离的最小值，最大值。

函数empolarcolors.m的作用是画出给定横截面位置直螺线管磁场的色彩映射图，参数设置与函数empolar.m相同。

函数emfixpd.m的作用是画出随直螺线管长度的增加，某一定点磁感应强度大小随直螺线管长度的变化图。R代表直螺线管半径，I为电流，d为螺距，L0为直螺线管最小长度，dL为直螺线管长度变化的步长，N1为取不同直螺线管长度的个数，N为每匝取的电流元个数。坐标系中直螺线管的位置为：轴线与z轴重合，xOy平面为螺线管底端，螺线管在z轴的投影为0<=z<=L（L为螺线管长度）。（x，y，z）为给定点的坐标。

函数emfixpL.m的作用是画出随直螺线管螺距的减小，某一定点磁感应强度大小随直螺线管螺距大小的变化图。R代表直螺线管半径，L为螺线管长度，I0表示直螺线管的面电流（直螺线管电流I=I0*d（d为螺线管螺距）），d0为最大螺距，dd为螺距变化的步长，N1为取不同直螺线管螺距的个数，N为每匝取的电流元个数。坐标系中直螺线管的位置为：轴线与z轴重合，xOy平面为螺线管底端，螺线管在z轴的投影为0<=z<=L。（x，y，z）为给定点的坐标。

函数emmidpd.m的作用是画出随直螺线管长度的增加，直螺线管中部横截面上某定点磁感应强度大小随直螺线管长度的变化图。R代表直螺线管半径，I为电流，d为螺距，L0为直螺线管最小长度，dL为直螺线管长度变化的步长，N1为取不同直螺线管长度的个数，N为每匝取的电流元个数。坐标系中直螺线管的位置为：轴线与z轴重合，xOy平面为螺线管底端，螺线管在z轴的投影为0<=z<=L（L为螺线管长度）。（x，y，L/2）为给定点的坐标。

函数emmidpL.m的作用是画出随直螺线管螺距的减小，直螺线管中部横截面上某定点磁感应强度大小随直螺线管螺距大小的变化图。R代表直螺线管半径，L为螺线管长度，I0表示直螺线管的面电流（直螺线管电流I=I0*d（d为螺线管螺距）），d0为最大螺距，dd为螺距变化的步长，N1为取不同直螺线管螺距的个数，N为每匝取的电流元个数。坐标系中直螺线管的位置为：轴线与z轴重合，xOy平面为螺线管底端，螺线管在z轴的投影为0<=z<=L。（x，y，L/2）为给定点的坐标。

函数emmidfuncd.m的作用是画出随密绕直螺线管长度的增加，直螺线管中部横截面上某距轴距离与螺线管长度有关的点的磁感应强度大小随密绕直螺线管螺距大小的变化图。R代表直螺线管半径，I为电流，L0为直螺线管最小长度，dL为直螺线管长度变化的步长，N1为取不同直螺线管长度的个数，N为每匝取的电流元个数。x_func为关于L和R的函数句柄，为考察螺线管外场的一致收敛性，一般取x_func关于L单调递减趋于R，故需要保证直螺线管的近似密绕性——取d0为最大螺距，dd为螺距变化的步长，直螺线管螺距d=d0-(L-L0)/dL*dd。坐标系中直螺线管的位置为：轴线与z轴重合，xOy平面为螺线管底端，螺线管在z轴的投影为0<=z<=L。（x_func（L，R），0，L/2）为参考点的坐标。

函数emverticalstream2.m的作用是画出直螺线管的过轴纵截面的磁力线图和磁场大小的色彩映射图。在函数的参数中，R代表直螺线管半径，I为电流，L为长度，d为螺距，N为取直螺线管的总电流元数，N1为采样点个数的平方根。坐标系中直螺线管的位置为：轴线与z轴重合，xOy平面为螺线管底端，螺线管在z轴的投影为0<=z<=L。采样在平面{y=0,h1<=z<=h2,-s<=x<=s}区域内进行。为剔除极少数磁场大小极大的瑕点，画色彩映射图时剔除磁感应强度大小在95%位数以上的点。

函数emcone2.m的作用是画出锥形螺线管的过轴纵截面的磁力线图和磁场大小的色彩映射图。在函数的参数中，R1，R2为最小，最大半径，I为电流，L为长度，d为螺距，N为取锥形螺线管的总电流元数，N1为采样点个数的平方根。坐标系中锥形螺线管的位置为：轴线与z轴重合，xOy平面为螺线管半径R1的一端，螺线管在z轴的投影为0<=z<=L。采样在平面{y=0,h1<=z<=h2,-s<=x<=s}区域内进行。为剔除极少数磁场大小极大的瑕点，画色彩映射图时剔除磁感应强度大小在95%位数以上的点。

函数emdual2.m的作用是画出双线圈系统产生的过轴截面的磁力线图和磁场大小的色彩映射图。在函数的参数中，R1，R2为两个螺线管各自的半径，I为电流，L为长度，d为螺距，D为两个螺线管距离的二分之一，N为取的总电流元数，N1为采样点个数的平方根。坐标系中两个螺线管的位置为：轴线与z轴重合，半径R1的螺线管在z轴的投影为-L-D<=z<=-L，半径R2的螺线管在z轴的投影为L<=z<=L+D。采样在平面{y=0,h1<=z<=h2,-s<=x<=s}区域内进行。为剔除极少数磁场大小极大的瑕点，画色彩映射图时剔除磁感应强度大小在95%位数以上的点。
